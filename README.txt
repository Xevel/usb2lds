USB2LDS
=======

2014 Nicolas "Xevel" Saugnier

Interface boards for the Neato LDS - the Lidar of the XV-11.
More on the Neato LDS : http://xv11hacking.wikispaces.com/LIDAR+Sensor


Boards:
- USB2LDS: USB interface for the Neato LDS, that also takes care of the motor control.
- Neato LDS Breakout Board (or NLDSBB): Simple breakout board that exposes the UART of the LDS, but takes care of the motor control.
